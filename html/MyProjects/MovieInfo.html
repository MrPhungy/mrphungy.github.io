Visualizing Movie Data

For a bit of fun, curiosity, and self-learning, I wanted to display all my movie data from my Letterboxd account. However, this proved to be difficult considering that Letterboxd does not offer a public API quite yet. So, I wrote a Python script to scrape my data from Letterboxd and save it into a Google Sheet. The Python script is run automatically once every 24 hours, ensuring new data daily. With the power of Looker Studio, the data is then sliced and diced, allowing me to dynamically create this neat little chart to display all the information.

*If you're having trouble viewing the chart, please log into your Google account and enable third-party cookies. Or, click here!
