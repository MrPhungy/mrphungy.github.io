A Look Into My Movies

Visualizing Movie Data

*If viewing on mobile device, please view in landscape mode for now. An update will come in the future.

For a bit of fun, curiosity, and self-learning, I wanted to display all my movie data from my Letterboxd account. However, this proved to be difficult considering that Letterboxd does not offer a public API quite yet. So, I wrote a Python script to scrape my data from Letterboxd and save it into a CSV file. With the power of Tableau, the data is then sliced and diced allowing me to create this neat little viz to display all information.

I am currently working on dynamically updating the data.
